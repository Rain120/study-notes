#### 编辑相关技巧

**Note:** `Mac`展示，`window`待总结

```
command ⌘
option ⌥ (alt)
ctrl ⌃
```



- 光标移动

  ```
  // 单词移动
  option(⌥) + ➡️
  option(⌥) + ⬅️
  
  // 行首行尾移动
  command(⌘)+ ➡️
  command(⌘) + ⬅️
  
  // 单词分解移动
  ctrl(⌃) + option(⌥) + ➡️
  ctrl(⌃) + option(⌥) + ⬅️
  ```

  ![cursor-op](./images/cursor-op.gif)

- 行 - 复制、移动、删除、合并、插入、缩进

  ```
  1. 单行
      复制
      command + c
      shift + option + ⬆️
      shift + option + ⬇️
  
      移动
      option + ⬆️
      option + ⬇️
  
      删除
      command + x
  
      合并
      ctrl + j
  
      插入
      在上面插入: shift + command + enter
      在下面插入: command + enter
  
      缩进
      向左缩进: command + [
      向右缩进: command + ]
  2. 多行
    选择: option + 鼠标选择
  
  ```

  **单行**

  ![line-op](./images/line-op.gif)

  **多行**
