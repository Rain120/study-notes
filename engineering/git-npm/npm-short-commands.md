#### 短命令

- `-v`: `--version`
- `-h`, `-?`, `--help`, `-H`: `--usage`
- `-s`, `--silent`: `--loglevel silent`
- `-q`, `--quiet`: `--loglevel warn`
- `-d`: `--loglevel info`
- `-dd`, `--verbose`: `--loglevel verbose`
- `-ddd`: `--loglevel silly`
- `-g`: `--global`
- `-C`: `--prefix`
- `-l`: `--long`
- `-m`: `--message`
- `-p`, `--porcelain`: `--parseable`
- `-reg`: `--registry`
- `-f`: `--force`
- `-desc`: `--description`
- `-S`: `--save`
- `-P`: `--save-prod`
- `-D`: `--save-dev`
- `-O`: `--save-optional`
- `-B`: `--save-bundle`
- `-E`: `--save-exact`
- `-y`: `--yes`
- `-n`: `--yes false`
- `ll` and `la` commands: `ls --long`
