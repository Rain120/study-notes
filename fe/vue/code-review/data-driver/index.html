1. VDom​

为什么需要使用虚拟DOM？

浏览器是解析HTML先生成一个DOM树，后面生成CSSOM树，合并成Render树，然后进行布局，回流重绘操作，如果频繁的DOM操作，会不断的带来浏览器的回流与重绘，当然，浏览器会通过JS来操作对象，因为JS的性能比较好，性能上面优于真实DOM。
