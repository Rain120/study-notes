| Hooks                                      | Class Componets                          |
| ------------------------------------------ | ---------------------------------------- |
| `useEffect`                                | `componentDidMount`,`componentDidUpdate` |
| `useEffect` 第一个参数`return`的`callback` | `componentWillUnmount`                   |
| 函数没有                                   | `constructor`                            |
| 函数组件体本身                             | `render`                                 |
| `React.memo`                               | `shouldComponentUpdate`                  |
| `useState` `update`函数                    | `getDerivedStateFromProps`               |
